🚗 Car Price Prediction using Machine Learning

📌 Project Overview

This project predicts the selling price of a used car using Machine Learning. It analyzes vehicle features such as manufacturing year, fuel type, transmission, kilometers driven, seller type, and ownership history to estimate the expected market price.Objective

Predict used car prices accurately.

Compare different regression algorithms.

Deploy the trained model with a Streamlit web application.
